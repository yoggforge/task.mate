To-Do list extension

I built this extension because I was genuinely fed up with bloated productivity tools. Too many features, endless settings, sign-ins, syncing—just way too much. All I wanted was a simple space to write down what I need to do and move on with my day. So I made one. No fluff, no learning curve. Just open the extension, type your task, and you're done. That’s it. It’s productivity without the overhead.

Task.mate is a minimal offline to-do list Chrome extension. Stay focused and organized without leaving your browser.

  • Edit, reorder, check, delete, and restore tasks easily.
  • Minimal distraction UI embedded in your Chrome.
  • Fully offline, secure, and fast.

This free plugin is designed with simplicity in mind, offering all the core features you need to get started quickly, without any unnecessary clutter.

Feature Description
Add Task Create new tasks in your to-do list.
Edit Task Change task content after adding it.
Check Task Mark tasks as completed or done.
Delete Task Remove tasks from your list.
Clear All Tasks Erase all tasks at once.
Reorder Tasks Prioritize tasks using drag and drop to change their order.
